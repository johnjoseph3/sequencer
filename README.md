# Sequencer

Simple ES6 drum sequencer utilizing the [Web Audio API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API).

## Development

Run webpack-dev-server on port 8080:

```sh
$ npm run start-dev
```

## Production
Run production on port 3000:

```sh
$ webpack
$ npm run start-prod
```
## Credits
